# Dtached Tournament Platform

Monorepo for the Dtached football tournament platform.

## Structure

```
├── frontend/    React + Vite (port 5173)
├── backend/     Spring Boot API (port 8080)
```

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
mvn spring-boot:run
```
